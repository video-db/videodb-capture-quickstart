# Changelog

## [2.0.0] - 2026-02-05

### Changed
- **SDK Migration**: Migrated from Recorder SDK to **Capture SDK** (`videodb/capture`)
- **Tunneling**: Replaced ngrok/localtunnel with **Cloudflare Quick Tunnels** (zero-config)
- **Webhook Events**: Updated to use `capture_session.exported` event format

### Features
- Real-time transcription via WebSocket connections
- Live video preview with RTSP relay
- AI-powered meeting insights generation
- Recording history with playback
